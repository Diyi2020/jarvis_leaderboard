# Model for Band_gap_HSE
<!--benchmark_description--> - Description: This is a benchmark to evaluate how accurately an AI model can predict the band gap at the accuracy level of HSE06 (hybrid DFT functional) using the snumat dataset. The dataset contains different types of chemical formula and atomic structures. Here we use mean absolute error (MAE) to compare models with respect to DFT (HSE06) accuracy.
