# Model for gappbe
<!--benchmark_description--> - Description: This is a benchmark to evaluate how accurately an AI model can predict the bandgap using the M3GNET dataset (PBE functional). The dataset contains different types of chemical formula and atomic structures. Here we use mean absolute error (MAE) to compare models with respect to DFT (PBE) accuracy. External links: https://github.com/materialsvirtuallab/m3gnet
