# Model for exfoliation_energy
<!--benchmark_description--> - Description: This is a benchmark to evaluate how accurately an AI model can predict the energy above the exfoliation energy (isolating a monolayer from a layered structure) using the JARVIS-DFT (dft_3d) dataset. The dataset contains different types of chemical formula and atomic structures. Here we use mean absolute error (MAE) to compare models with respect to DFT accuracy.
