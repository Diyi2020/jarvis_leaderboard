# Model for slme
<!--benchmark_description--> - Description: This is a benchmark to evaluate how accurately an AI model can classify a material as a solar cell based on the DFT computed Spectroscopic Limited Maximum Efficiency (SLME) from the JARVIS-DFT (dft_3d) dataset. The dataset contains different types of chemical formula and atomic structures. Here we use accuracy of classification (ACC) to compare models with respect to DFT accuracy.
